Bluet My Favicon let you easely change the favicon of your site.

== Description ==
"BleuT My FavIcon" plugin allows you simply customise your site's favicon by uploading an image file.

== Installation ==
1. Download the plugin
1. Upload the bluet-favicon folder to your site's wp-content/plugins directory
1. Activate the plugin in the WordPress Admin Panel
1. in the "Appearance" menu choose "My Favicon" ...
